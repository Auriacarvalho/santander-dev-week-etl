# Santander Dev Week 2023 – ETL com Python

## Descrição
Projeto desenvolvido como desafio da DIO, implementando um pipeline ETL
(Extract, Transform, Load) com Python.

Devido à indisponibilidade da API pública do evento, o processo utiliza
dados simulados, mantendo toda a arquitetura e fluxo de ETL.

## Tecnologias
- Python
- Pandas

## Pipeline ETL
- Extract: Leitura de dados a partir de CSV e dados simulados
- Transform: Enriquecimento com mensagens personalizadas
- Load: Simulação de carga dos dados processados

## Execução
```bash
python etl.py
