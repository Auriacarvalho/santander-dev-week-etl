# Santander Dev Week - ETL

## 📌 Descrição
Este projeto foi desenvolvido como parte da **Santander Dev Week**, com foco em **processos ETL (Extract, Transform, Load)**.  
O objetivo é construir um pipeline de dados que realiza a extração de informações, transformação para padronização e carregamento em um banco de dados, permitindo análises mais eficientes.

## 🚀 Tecnologias Utilizadas
- **Python** → scripts de ETL  
- **Pandas / NumPy** → manipulação e transformação de dados  
- **SQL** → carregamento e consultas no banco de dados  
- **Jupyter Notebook** → documentação e execução dos processos  
- *(adicione outras ferramentas usadas, ex.: Airflow, AWS, etc.)*

## 📂 Estrutura do Projeto
- `data/` → bases de dados de entrada  
- `src/` → scripts de ETL (extração, transformação e carga)  
- `notebooks/` → análises exploratórias e validação  
- `docs/` → documentação e anotações  

## 📊 Resultados
- Pipeline ETL funcional para ingestão e transformação de dados.  
- Estrutura organizada para facilitar manutenção e escalabilidade.  
- Relatórios e consultas SQL para análise dos dados carregados.
